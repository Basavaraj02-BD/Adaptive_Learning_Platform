# 🎓 Adaptive Learning Platform with Auto Evaluation

## 📌 Project Overview

The **Adaptive Learning Platform** is a smart web-based application built using **Python & Django** that personalizes learning for students and provides automatic evaluation of their performance.

It helps learners improve efficiently by adapting content based on their understanding level and gives instant feedback.

---

## 🚀 Features

* 👨‍🎓 Student Registration & Login System
* 📚 Course & Content Management
* 🧠 Adaptive Learning (based on performance)
* 📝 Auto Evaluation System
* 📊 Performance Tracking Dashboard
* 🔐 Secure Authentication System

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Python (Django Framework)
* **Database:** MySQL
* **Tools:** PyCharm, XAMPP

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/Adaptive_Learning_Platform.git
```

2. Navigate to project folder:

```
cd Adaptive_Learning_Platform
```

3. Create virtual environment:

```
python -m venv .venv
```

4. Activate environment:

```
.venv\Scripts\activate
```

5. Install dependencies:

```
pip install -r requirements.txt
```

6. Run migrations:

```
python manage.py migrate
```

7. Run server:

```
python manage.py runserver
```
